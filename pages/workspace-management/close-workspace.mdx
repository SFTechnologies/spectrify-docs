## Close workspace
<hr></hr>
<br></br>
<Image src={`${imagePath}close_workspace_icon.svg`} alt="close workspace icon" width={65} height={65} />

Close workspace option terminates the current workspace. 

Any unsaved data within the workspace will be lost. This action effectively terminates the current workspace session, closing all associated documents and results. Users should ensure that any necessary data is saved before proceeding with the closure of the workspace.
