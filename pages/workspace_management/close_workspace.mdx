## Close workspace
<hr></hr>
<br></br>
<Image src="/images/close_workspace_icon.svg" alt="Close workspace icon" width={50} height={50} />

Close workspace option ends the current workspace. 

Any unsaved data within the workspace will be lost. This action effectively ends the current workspace session, closing all associated documents and results. Users should ensure that any necessary data is saved before proceeding with the closure of the workspace.
