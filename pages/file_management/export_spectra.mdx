## Export spectra
<hr></hr>
<br></br>
<Image src="/images/export_spectra_icon.svg" alt="export spectra icon" width={50} height={50} />

Allows users to export spectra and their associated metadata in .spsp format. The number in parentheses denotes the count of selected spectra. 

By using the import pop-up, users can conveniently name the spectra and export them through the Operating System's file menu.
