## Delete spectra
<hr></hr>
<br></br>
<Image src="/images/delete_spectra_icon.svg" alt="delete spectra icon" width={50} height={50} />

This tool allows users to delete selected spectra and their metadata from the workspace. The number in parentheses indicates the count of selected spectra. 

Please note that this action is irreversible, and the deleted data cannot be recovered.
